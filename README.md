RapidReport/dotnet
==================
RapidReport/dotnet は .NET 用の帳票レンダリングエンジンです。

JSON 形式で記述した帳票定義ファイルを読み込み、データを渡すことで帳票を出力します。

RapidReport  
[http://rapidreport.systembase.co.jp/](http://rapidreport.systembase.co.jp/)

動作環境
-------
.NET Framework 3.5 以上（Visual Studio 2010 プロジェクト）
